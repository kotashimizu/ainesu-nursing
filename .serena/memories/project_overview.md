# プロジェクト概要: あいねすの家訪問看護ステーション

## プロジェクトの目的
愛知県豊橋市の訪問看護ステーション「あいねすの家」の公式ウェブサイト。
アイデアルサポート株式会社が運営する訪問看護サービスの情報提供と問い合わせ受付を目的とした静的サイト。

## 主な機能
- 訪問看護サービスの紹介（小児・精神科に特化）
- 会社概要の表示
- サービス利用の流れ説明
- お問い合わせフォーム
- レスポンシブ対応
